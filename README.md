# WestminsterSim — Landing Page

Marketing/landing site for **WestminsterSim**, a mobile-first, text-driven UK political
career simulator.

Static site, no build step:

- `index.html` — landing page
- `privacy.html` — privacy policy
- `styles.css` — landing-page styles
- `assets/` — logo, favicon, banner, and gameplay screenshots

## Local preview

```bash
python3 -m http.server 8000
# then open http://localhost:8000/
```

## Deploy

Deploys as-is on Vercel (or any static host) — no framework or build command required.
